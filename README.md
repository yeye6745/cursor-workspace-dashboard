# Dashboard Prototype

본 프로젝트는 개발자가 UI/UX 동작과 기획 의도를 이해할 수 있도록 설명하기 위한 프로토타입 제작을 목적으로 합니다.

## 프로젝트 목적

- 실제 개발 산출물이 아니므로, 구현 완성도나 코드 품질보다는 디자인 요소와 UI 구조가 의도한 위치와 형태로 명확히 표현되는 것을 우선합니다.
- Figma 화면을 시각적으로 최대한 동일하게 구현합니다.
- UX적인 동작에 대해서는 사용자의 입력(클릭, 호버, 드래그 등)을 기반으로 기획 의도가 전달될 수 있도록 해석하여 구성합니다.

## 기술 스택

- **React**: UI 라이브러리
- **Vite**: 빌드 도구 및 개발 서버

## 이미지 리소스

Figma 자동 생성 코드에 포함된 `figma:asset` 형태의 이미지 리소스는 실제 개발 환경에서 사용하지 않으므로, 프로토타입 구현 시 `src/assets/figma/icon` 경로에 위치한 로컬 이미지 파일로 치환하여 사용합니다.

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```
